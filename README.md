DİJİTAL YOKLAMA SİSTEMİ

Projenin Amacı:
Öğrenci, ders, yoklama oturumu ve yoklama kayıtlarının yönetilmesini sağlayan bir uygulamadır.

Kullanılan Teknolojiler:
- C#
- Windows Forms
- Entity Framework
- SQL Server

Modüller:
- Öğrenci İşlemleri (Ekle, Sil, Listele)
- Ders İşlemleri (Ekle, Sil, Listele)
- Yoklama Oturumu Yönetimi
- Yoklama Kayıtları Yönetimi

Veritabanı Tabloları:
- Ogrenciler
- Dersler
- YoklamaOturumlari
- YoklamaKayitlari
